## 📌 Project Overview
This project is based on understanding and applying the concept of digital literacy in real life. It includes different tasks that helped me learn how to use online platforms, communicate professionally, and stay safe in the digital world.

---

## 🎯 Objectives
- To understand the importance of digital literacy  
- To create and manage online profiles  
- To learn safe and responsible internet usage  
- To improve communication through emails  
- To understand cybercrime and its prevention  

---

## 📂 Tasks Included

### 🔹 Task 1: Presentation
Created a presentation on digital literacy using Canva.

### 🔹 Task 2: Portfolio
Created and managed professional profiles on LinkedIn and GitHub.

### 🔹 Task 3: Platforms
Used platforms like Canva and Google Forms to create content and surveys.

### 🔹 Task 4: Email Etiquette
Learned how to write formal emails and understood do’s and don’ts of online communication.

### 🔹 Task 5: Cybercrime
Studied cybercrime, its impact, and created a case study with prevention methods.
